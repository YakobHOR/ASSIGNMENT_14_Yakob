# ASSIGNMENT_14_Yakob
This project explores ethical machine learning, focusing on fairness, bias detection, and model explainability. Using the UCI Adult Income dataset, we train a logistic regression classifier to predict whether an individual earns more than $50K per year.

Beyond standard model evaluation, the project integrates:

Fairness analysis using Fairlearn

Global explainability using SHAP

Local explainability using LIME

The goal is to demonstrate how machine learning systems can be evaluated not only for accuracy but also for equity, transparency, and ethical impact.
